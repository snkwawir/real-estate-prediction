# Taiwan Real Estate Price Prediction Models

## Overview
This project develops multiple regression-based machine learning models to predict residential property valuations using structured housing transaction data from Taiwan’s real estate market. The analysis compares traditional linear regression, stepwise feature selection, and Lasso regularization approaches while evaluating predictive accuracy, interpretability, and statistical validity.

## Objectives
- Predict residential property prices using structured housing datasets
- Compare regression modeling and regularization techniques
- Evaluate predictive accuracy across multiple modeling approaches
- Conduct statistical diagnostics and validation testing
- Identify the most influential factors affecting housing prices

## Tools & Technologies
- R
- glmnet
- MASS
- caret
- ggplot2
- Linear Regression
- Lasso Regression
- Statistical Diagnostics

## Key Features
- Built traditional linear regression, stepwise selection, and Lasso regression models
- Applied cross-validation workflows to optimize regularization parameters
- Conducted residual analysis, VIF testing, and Durbin-Watson validation
- Compared RMSE and R² performance across multiple predictive models
- Evaluated model interpretability versus predictive performance trade-offs

## Results
The Lasso regression model achieved the strongest predictive performance while maintaining strong generalization capability. The project demonstrated how regularization and feature selection techniques can improve model stability and predictive reliability in real estate valuation workflows.
